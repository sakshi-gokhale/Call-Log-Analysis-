# 📞 Call Log Analysis Dashboard

## Overview

Developed an interactive Power BI dashboard using a CSV dataset to analyze call logs and monitor communication performance. The dashboard provides insights into incoming, outgoing, connected, missed, rejected, and unattended calls through visual reports and KPI metrics.

## Features

* Incoming and outgoing call analysis
* Missed, rejected, and unattended call tracking
* Employee-wise performance analysis
* Monthly connected call trends
* Region-wise call distribution
* Interactive KPI dashboards

## Tools Used

* Power BI
* CSV Dataset
* Data Cleaning & Transformation
* Data Visualization

## Key Insights

* Identified employees with the highest missed and unattended calls.
* Analyzed monthly call connectivity trends.
* Compared incoming and outgoing calls across years.
* Evaluated call distribution across regions.
